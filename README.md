````bash
1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/shoppe.git
   cd shoppe
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

   ```
   http://localhost:3000
   ```
````
